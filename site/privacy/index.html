Marker Privacy Policy

Last updated: July 13, 2026

The short version

Marker collects nothing. There are no analytics, no telemetry, no accounts, no ads, and no third-party SDKs that phone home.

Your selections and clipboard

Text you select is stored only on your Mac, in a local file (~/Library/Application Support/Marker). It is never transmitted anywhere. Deleting the app or pressing Clear removes it.

Accessibility permission

Marker asks for macOS Accessibility access solely to read the text you select and to paste on your behalf. That access is used for nothing else.

Network

Marker connects to the internet for exactly one thing: checking for app updates via the standard Sparkle framework against getmarkerapp.net. An update check is a plain HTTPS request; like any web request it exposes your IP address and app version to the server, which we don't log or profile. No other network requests are made.

Changes

If this policy ever changes, the update will be published on this page.
